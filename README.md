# plang-fe


## 사전과제 : 알고리즘

### algorithms 결과 테스트케이스를 테스트 해볼 수 있도록 구축 해 두었습니다.

- **1 폴더 이동**
  - `cd ./plang/algorithms`

- **2 의존성 설치**
  - `pnpm install`

- **3 알고리즘 파일 실행 (번호로 실행)**
  - `pnpm dev -- 1` → `1.js` 실행
  - `pnpm dev -- 2` → `2.js` 실행
  - `pnpm dev -- 3` → `3.js` 실행


## 사전과제 : 타이머 구현

### feature: 타이머 React 초기세팅

- **사용 스택**
  - React + TypeScript + Vite
  - TailwindCSS (v4) + `@tailwindcss/vite`
  - UI: `@youngduck/yd-ui`
  - 상태관리: `zustand`
  - 품질: ESLint(flat config) + Prettier + Vitest
  - 테스트 코드 : `Vitest`

- **1 폴더 이동**
  - `cd ./plang/react`

- **2 의존성 설치**
  - `pnpm install`

- **3 개발 서버 실행**
  - `pnpm dev`
